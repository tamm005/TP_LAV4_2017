Explicación del juego ¿Qué ves?

¿Qué ves? - EL objetivo de este juego es muy simple: cada imagen representa una palabra o frase que debes descubrir.

Puede ser un objeto, un verbo, un adjetivo, etc. ¿Crees poder hacerlo?.
